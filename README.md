## To run this app

Clone the project

```bash
  git clone https://github.com/samueldusek/meetups-react-nextjs-app.git
```

Go to the project directory

```bash
  cd meetups-react-nextjs-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
